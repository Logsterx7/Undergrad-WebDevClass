Phase 0: Requirements Specification (10%)

This Program is challenging our skills with HTML and CSS. It is aimed at us to write webpages and balance CSS with HTML and also recreating files with and without the wonders of CSS. We will be required to know the basics of both HTML and CSS. Along with tying everything together nicely to act as a real website that you would see out in the internet.

This blog is going to showcase all the users hobbies and personalities while also showcasing some very helpful tech tips as well.

Phase 1: System Analysis (10%)

Here there isn't much data other than the web pages themselves. The only other data we have to keep track of is paths of the other pages and images.

Phase 2: Design (30%)

The idea here is to create a neatly organized system of webpages using CSS and HTML. the pages here of course are going to be the about.html, index.html, plan.html, style.css, techtips+css.html, and techtips-css.html. There will be a box at the top that will be used to navigate between pages simply. Each link will be displayed on top.

The input of this program will be the user clicking links to send them to other pages on the blog. The output of course just being the different pages that will display.

Phase 3: Implementation (15%)

The implementation of this will just be to simply go through and write it file by file. I will probably focus on one file at a time rather than going through the entire thing at once. starting from the index and then going from there. After HTML has been written then I plan on going back and implementing the CSS part.

Phase 4: Testing & Debugging (30%)

For testing. There are two main things that need to be done. The first is to check back and forth to not only make sure all the requirements have been met for this site, but to also to ensure that all of the files have zero errors. That part will be done through W3C;s Markup Validation Service and W3C's CSS validator.

One had to do with Alt tags. I remembered going over that in class a while ago, but it had slipped my mind. But overall I think it went by smoothly.

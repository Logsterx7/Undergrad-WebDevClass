HTML Elements

<title>

  • Names the tab that is in the browsers bar.
  • Belongs in <head> tag so it is neither block-level or inline.
  • Only contains text.

<style>

  • Holds information about how to style elements.
  • Neither block-level or inline since it holds everything
  • You can use multiple style elements.

<html>

  • Is the root element that holds everything.
  • style element should only be used in the <head> tag so it is neither block-level or inline

<div>

  • Treats a group of items as one rather than dealing with the items individually.
  • This item will take up as much space as it can, so it is a block-level.
  • Makes styling much easier.

CSS Selectors

.class

Setting up a class in CSS is simply done by putting a "." along with your class name. From there you can define all of its attributes inside curly braces after. Then when assignming an element to it all you need to do is type in its tag style = "yourClassName". This will allow you to style things the same way in one place

element

By typing the element such as p, h1, h2, etc, it allows you to style everything with that tag in your file.

#id

This will select every element that is labeled with the that id which is set to your id name. This is set up by setting a tags id by typing id="youidnamehere".

element.class

This will allow you to select all tags with that same element and class. Class is simply set by typing class="yourclassnamehere"

CSS Properties

background-color

Allows you to set color to the background of your elements. There are many values this can be set to. From names of colors to hex code values to get that color just right

color

This works the exact same way as background-color. The only difference is this one is specifically for text.

padding

This property allows you to add space to your elements. It's especially handy when working with block-level elements. Just type an interger with px after and itll create that many pixels of space around its boarder
